1. HTML

Volver al index

HTML (HyperText Markup Language, o Lenguaje de Marcado de Hipertexto) es el lenguaje estándar utilizado para crear y estructurar páginas web en la World Wide Web (WWW). Es el esqueleto de cualquier página web, y su principal función es definir la estructura y el contenido de un documento web. En términos sencillos, HTML es un conjunto de etiquetas (o "tags") que indican al navegador web cómo debe mostrar el contenido. Por ejemplo, se utiliza para estructurar texto, insertar imágenes, enlazar a otras páginas web, y mucho más.




2. CSS

Volver al index

CSS (Cascading Style Sheets, o Hojas de Estilo en Cascada) es un lenguaje de diseño que se utiliza para definir el estilo visual y la disposición de los elementos en una página web. En otras palabras, CSS es lo que le da el aspecto y el formato a una página HTML, permitiendo controlar cosas como colores, fuentes, márgenes, espaciado, alineación, animaciones, y otros aspectos del diseño visual.




3. Javascrip

Volver al index

JavaScript es un lenguaje de programación ampliamente utilizado para crear interactividad y dinamismo en las páginas web. A diferencia de HTML (que se encarga de la estructura)y CSS (que maneja el diseño y estilo), JavaScript se encarga de la interactividad y el comportamiento de la página.
